# PDFs

Store PDF reference files here.

Use this folder for lightweight project documents that Qwen, Open WebUI, Codex, or OpenClaw agents may need to reference.

Guidelines:

- Keep PDFs reasonably small.
- Do not store private, sensitive, or credential-bearing PDFs in this public repo.
- Do not store model files here.
- For very large PDFs, use external storage and place a link or note in Markdown memory instead.
